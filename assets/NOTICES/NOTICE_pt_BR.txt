Aviso de Licenças e Atribuições de Software de Terceiros
Projeto: AGENDA DE AVALIAÇÕES ACADÊMICAS
Versão do documento: 09 de Outubro de 2025

Este documento lista as principais bibliotecas e componentes de terceiros utilizados no projeto AGENDA DE AVALIAÇÕES ACADÊMICAS e fornece referências às suas licenças. O objetivo é cumprir requisitos de distribuição e oferecer informações claras para usuários, integradores e revisores. As dependências do projeto estão listadas no arquivo requirements.txt incluído no repositório/instalador.

Observação geral
- O produto final pode ser distribuído como executável Windows (build gerado por ferramentas como Nuitka). As bibliotecas Python listadas em requirements.txt são empacotadas no binário conforme a build.
- Os textos completos das licenças e os avisos de copyright de terceiros são fornecidos no pacote de distribuição/instalador (arquivos LICENSE, LICENSES, NOTICE ou diretório equivalente). Para obter cópias do código-fonte exigidas por licenças (por exemplo LGPL) contacte o autor conforme abaixo.

Tratamento das licenças e textos completos
- Os textos completos das licenças de componentes de terceiros, avisos de copyright e quaisquer notices exigidos são incluídos no pacote de distribuição/instalador (arquivos LICENSE, LICENSES, NOTICE ou equivalente).
- Em caso de necessidade de fornecer código-fonte de componentes licenciados sob LGPL (ou licença similar que exija disponibilização de código), siga as instruções de cada licença — contate o autor para coordenar fornecimento de fontes ou links.

Declaração sobre transmissão de dados e privacidade
- O executável padrão do projeto não envia telemetria automática a servidores externos.
- Todos os dados do usuário (por exemplo, listas de atividades, arquivos JSON de preferências, exportações PDF) são armazenados localmente no diretório persistente do usuário (por exemplo, %APPDATA% no Windows).
- Quaisquer integrações opcionais que impliquem transmissão de dados serão documentadas e requererão consentimento explícito do usuário.
- Para detalhes sobre tratamento de dados pessoais e privacidade, consulte a Política de Privacidade incluída no pacote (assets/PRIVACY_POLICY/Privacy_Policy_pt_BR.txt).

Notas sobre substituição de bibliotecas (quando aplicável)
- Bibliotecas vinculadas dinamicamente (ex.: PySide6 / Shiboken6) podem, em builds específicos, ser substituídas por versões compatíveis seguindo as instruções de empacotamento. A substituição deve respeitar as obrigações da licença (por exemplo, fornecimento de fontes quando requerido pela LGPL).
- Qualquer modificação ou substituição de bibliotecas em binários distribuídos deve acompanhar os avisos e textos de licença necessários.
